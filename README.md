## Развертка и запуск
Для развертывания сервиса требуется выполнить следующие действия:
```shell script
sudo git clone https://github.com/corvinus-cron/Web-dictionary.git

cd Web-dictionary

chmod +x start.sh

./start.sh
```
Для доступа к интерфейсу - [словарь](http://localhost/)
